## 📌 Overview

A collection of Jupyter notebook exercises completed as part of Microsoft Learn's Machine Learning fundamentals learning path. Covers core ML concepts including supervised learning, model evaluation, and data preprocessing.

**Tech Stack:** `Python` `Jupyter` `Scikit-learn` `Microsoft Learn` `ML Basics`

---

## 🚀 Getting Started

### Prerequisites

```bash
pip install -r requirements.txt
```

> Requires **Python 3.8+**

### Installation

```bash
git clone https://github.com/Onabanjomicheal/Microsoftlearn-training-ml-basics.git
cd Microsoftlearn-training-ml-basics
pip install -r requirements.txt
```

### Run

```bash
python main.py
```
